Задание
В двумерном N*K массиве целых чисел обнулить столбцы, в которых чётных элементов было больше чем нечётных. 
Алгоритм server
1.	Создание сокета с помощью функции socket(): Создается переменная server_fd, которой присваивается значение функции socket(), которая создает новый сокет и возвращает его дескриптор.
2.	Очистка структуры server_addr с помощью функции memset(): Структура server_addr очищается с помощью функции memset(), чтобы убедиться, что все ее поля будут инициализированы нулями.
3.	Заполнение структуры server_addr значениями AF_INET и PORT.
4.	Преобразование адреса сервера из текстового в бинарный формат с помощью функции inet_pton().
5.	Установление соединения с сервером с помощью функции connect().
6.	Вывод на экран исходных данных массива data с помощью функции print_data().
7.	Отправка данных массива data на сервер с помощью функции send().
8.	Получение данных от сервера с помощью функции read().
9.	Копирование полученных данных в массив data с помощью функции memcpy().
10.	Вывод на экран полученных данных массива data с помощью функции print_data().

Алгоритм client
1.	Создание сокета с помощью функции socket().
2.	Очистка структуры server_addr с помощью функции memset().
3.	Заполнение структуры server_addr значениями AF_INET и PORT: В структуру server_addr записываются значения AF_INET (для IPv4) и PORT (номер порта, на котором будет слушать сервер).
4.	Преобразование адреса сервера из текстового в бинарный формат с помощью функции inet_pton(): Адрес сервера, который был передан в виде текста, преобразуется в бинарный формат с помощью функции inet_pton() и записывается в поле sin_addr структуры server_addr.
5.	Установление соединения с сервером с помощью функции connect(): Если соединение установлено успешно, функция возвращает 0.
6.	Вывод на экран данных массива data с помощью функции print_data().
7.	Отправка данных массива data на сервер с помощью функции send(): Функция возвращает количество отправленных байтов.
8.	Получение данных от сервера с помощью функции read(): Функция возвращает количество прочитанных байтов.
9.	Копирование полученных данных в массив data с помощью функции memcpy().
10.	Вывод на экран полученных данных массива data с помощью функции print_data(): Функция print_data() выводит на экран содержимое массива data после обработки на сервере и передачи клиенту.
